# Интеллектуальная система обновления статических ресурсов для мобильных приложений

Проект посвящен разработке системы, которая позволяет обновлять статические данные в мобильном приложении без выпуска новой версии через App Store. 

В рамках работы мы реализовали клиентский SDK, серверную часть, механизм принятия решения об обновлении и демонстрационное iOS-приложение.
