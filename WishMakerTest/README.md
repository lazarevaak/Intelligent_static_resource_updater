# WishMaker-App

## Описание

**WishMaker-App** — это iOS-приложение для создания, хранения и планирования исполнения желаний. Пользователь может добавлять желания, редактировать их, отслеживать, а также планировать их выполнение с помощью встроенного календаря.

---

## Основные возможности

- **Полностью программный UI** — UIKit и Auto Layout без использования Storyboard.
- **Архитектура MVVM** — чёткое разделение View, ViewModel и Model для упрощения поддержки и масштабируемости.
- **Локальное хранение** — UserDefaults для добавления, редактирования и удаления желаний.
- **Планирование событий** — интеграция с системным календарём iOS через **EventKit**.
- **Навигация** — использование `UINavigationController` для управления переходами между экранами.
- **Чистый код** — устранение "магических чисел" через `LayoutConstants`, использование `// MARK:` для структурирования кода.
- **Темы оформления** — передача цветовой темы между экранами с использованием биндингов.
